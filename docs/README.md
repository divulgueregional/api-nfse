# API-NFS-e

## Introdução

Essa documentação visa auxiliar a implementação com a API NFS-e Ambiente Nacional, disponibilizando um conjunto de funcionalidade que permitem acesso seguro as operações aqui disponíveis

## PRIMEIRO PASSSO
No ambiente de homologação você pode testar com qualquer cidade e CPF/CNPJ válido com certificado digital A1. Para produção apenas as cidades qie aderiram a emissão ao Ambiente Nacional.
